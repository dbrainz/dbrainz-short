DBrainz's FCC URL Shortener Service

Pass in a valid http URL in the format https://dbrainz-short.herokuapp.com/new/URL TO SHORTEN

Example: https://dbrainz-short.herokuapp.com/new/https://www.freecodecamp.com'

Example return : {"originalURL":"https://www.freecodecamp.com","shortURL":"https://dbrainz-short.herokuapp.com/1004"}

Passing in an invalid URL will return {"error":"invalid url"}

Entering https://dbrainz-short.herokuapp.com/1004 in a broswer will then redirect to https://www.freecodecamp.com
